## Installation

You can install Veve globally using npm:

<Tabs items={['npm', 'pnpm','yarn']}>
  <Tab value="npm">
   ```bash
   npm i -g veve
   ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm install veve
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn install veve
    ```
  </Tab>
</Tabs>

Alternatively, use our interactive setup wizard:

```bash title="Terminal"
npx veve-setup
```

### Writing Your First Test

Create a new TypeScript file and write your test:

```ts twoslash title="test.test.ts"
import "veve";

const add = (a: number, b: number): number => a + b;

it("should add two numbers", () => {
    assert(add(1, 2)).toBe(3)
});

run();
```
</Step>

<Step>

### Running Tests

Execute your tests using the `veve` command:

```bash title="Terminal"
veve
```
